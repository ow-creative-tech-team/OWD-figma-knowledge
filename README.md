# OWD Figma Knowledge Base

This repository is being organized as a multi-source knowledge base for OWD Figma Copilot.

Current content:

- [`OWD-figma-api-snapshot-master/`](OWD-figma-api-snapshot-master/) contains the Figma developer docs snapshot crawler and generated API/widget dataset.
- [`.github/workflows/daily-crawl.yml`](.github/workflows/daily-crawl.yml) runs the nested snapshot crawler on a daily schedule.

Additional knowledge sources can be added later as sibling top-level folders.
